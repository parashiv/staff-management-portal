Detailed Description

Creates a form for the Leave model, using the django ModelForm.

Used to take input the leave request details posted by the staff and store them in the Leave model. The author and approved field is kept hidden. Class Meta: An inner class to provide metadata to the ModelForm class. Provides an association between the ModelForm and the model Leave.
